Python 概述

Python的发展与应用领域

  • 大数据
  • 大模型
  • 人工智能

主流开发环境的特点及安装

  • 官网：https://www.python.org/
    • option –> Configure IDLE
    • IDLE –> new(进入代码编辑窗口) –> save –> run
    • 缺点：两个窗口，切换不方便，调试也不方便
  • Thonny: https://thonny.org/
    • Standard (not Raspberry PI)
    • 命令与代码为一体
  • PyCharm: https://www.jetbrains.com/pycharm/
    • 一般用于商业开发，环境比较复杂（功能比较齐全）
    • Community(free)
    • Professional
  • Thonny 树莓派典型嵌入的最主流开发工具(创新学分)

Python程序的特点与风格(Thonny)

  • 交互环境中的简单编程
  • 大小写敏感
  • 变量直接赋值使用(初始化)，无需指定类型
  • indentation：严格缩进，否则就是错误，根据缩进作为代码逻辑
  • 扩展名：py, pyw(窗体)

程序示例

  • 注释 # 开头，作为说明或者暂时屏蔽代码

  • 简单顺序程序（输入与输出）

  • map 函数的应用示例

开发环境强化，断点，调试

  • 视图 （View）–> Shell （交互命令行窗口）
  • 大纲(outlines)：查看函数
  • 变量：查看值
  • 行号前双敲两下，设置断点，运行，查看变量的值
    • 运行 –> 调试当前脚本(Ctrl + F5)

Tips

  • 专业见习、毕业实习、毕业设计，编程思维等等

作业要求

  • 标准：全学号_ 姓名_n
  • 按班为单位，备注未提交学生的学号和姓名
  • 包内无包
  • 3月9日晚上提交
